# 🚀 Quick Start: Get on Google in 30 Minutes

Follow these steps RIGHT NOW to get your site on Google!

---

## ⚡ Step 1: Google Search Console (15 minutes)

### 1.1 Create Account & Add Property

1. Go to: **https://search.google.com/search-console**
2. Sign in with your Google account
3. Click **"Add Property"**
4. Select **"URL prefix"**
5. Enter: `https://mercy-bett.vercel.app`
6. Click **"Continue"**

### 1.2 Verify Ownership

Google will show you a verification meta tag like this:

```html
<meta name="google-site-verification" content="xxxxxxxxxxx" />
```

**What to do:**

1. Copy the entire meta tag
2. Open `index.html` file in your project
3. Add it in the `<head>` section (around line 14, after the charset meta tag)
4. Save the file

### 1.3 Deploy to Vercel

```bash
# Commit your changes
git add .
git commit -m "Add Google verification"
git push origin main
```

Or simply push to GitHub - Vercel will auto-deploy!

### 1.4 Complete Verification

1. Wait 1-2 minutes for deployment
2. Go back to Google Search Console
3. Click **"Verify"**
4. ✅ Success! You're verified!

---

## ⚡ Step 2: Submit Sitemap (2 minutes)

Still in Google Search Console:

1. Click **"Sitemaps"** in left menu
2. In the box "Add a new sitemap", type: `sitemap.xml`
3. Click **"Submit"**
4. ✅ Done! Google will now crawl your site!

---

## ⚡ Step 3: Request Immediate Indexing (5 minutes)

1. Click **"URL Inspection"** in left menu
2. Type: `

https://mercylangat.com/` 3. Wait for Google to check 4. Click **"Request Indexing"** 5. Wait 1-2 minutes 6. ✅ Your homepage is submitted!

**Bonus:** Repeat for your book purchase page:

## 🆘 Need Help?

### **Common Issues:**

**"I can't find where to add the meta tag"**

- Open: `index.html` (in root folder)
- Look for `<head>` section
- Add the Google meta tag after line 6
- Before the `<!-- Primary Meta Tags -->` comment

**"Verification failed"**

- Make sure you deployed after adding the tag
- Wait 2-3 minutes and try again
- Check that the tag is in the `<head>` section
- Make sure you copied the ENTIRE tag

**"Site not showing on Google after 2 weeks"**

- Check Google Search Console for errors
- Make sure robots.txt is correct (it is!)
- Request indexing again
- Be patient - sometimes takes 4 weeks
